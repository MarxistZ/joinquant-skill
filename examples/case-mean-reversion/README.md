# 案例：RSI + 布林带均值回归策略

## 策略思路

对 2 只蓝筹股（平安银行、招商银行）使用布林带下轨 + RSI 超卖双重信号买入，上轨或超买时卖出。

## 用户原始需求

> "帮我写一个均值回归策略，交易平安银行和招商银行，用布林带和 RSI 指标判断超买超卖。"

1. **模板匹配**：`templates/05-mean-reversion.py`
2. **追问参数**：

   - 参数：使用默认（BB 20 日 2 倍标准差，RSI 14 日）
   - 仓位：每只最多 40%
3. **生成代码**：见 `strategy.py`
4. **lint 检查**：PASSED
