# 13 — 归因分析

> 聚宽回测结果页的归因分析功能，帮助理解策略收益的来源和风险暴露。

## 概览

归因分析在回测完成后自动生成，包含以下模块：

1. **净值分析** — 收益曲线及各维度分析
2. **Brinson 归因** — 配置 vs 选股贡献
3. **Fama-French 风格分析** — 五因子模型分解
4. **风险分析** — Barra 风险因子暴露

| 指标 | 说明 |

## Brinson 归因

基准为沪深300，采用减法超额。

| 归因项 | 含义 |
|---|---|
| **总超额收益** | 策略相对基准获得的额外收益，是下面三项的汇总 |
| **主动配置收益** | 对上涨行业超配或对下跌行业低配的贡献。>0 说明看准了市场大方向 |
| **标的选择收益** | 在同行业内选出更优个股的能力。>0 说明选股能力优于市场 |
| **互动收益** | 配置和选股共同作用的部分，即超额收益中同时受两者影响的部分 |

## Fama-French 五因子风格分析

将超额收益分解为 5 个因子：

| 因子 | 解释 | 系数解释 |
|---|---|---|
| **市场因子 (RM)** | 市场整体走势影响 | β > 0 与市场趋势一致; > 1 偏激进 |
| **规模因子 (SMB)** | 小盘股 vs 大盘股收益差 | s > 0 偏好小盘 |
| **估值因子 (HML)** | 高账面市值比 vs 低账面市值比 | h > 0 偏好价值股 |
| **盈利因子 (RMW)** | 高盈利 vs 低盈利公司 | r > 0 偏好高盈利 |
| **投资因子 (CMA)** | 低投资率 vs 高投资率 | c > 0 偏好低投资率公司 |

## 风险因子暴露分析（Barra）

10 个风格因子的暴露度与基准（沪深300）对比。基准风险暴露度为沪深300指数股票池中股票风险暴露的市值加权平均。

| 因子 | 说明 |
|---|---|
| 市值 (size) | 大盘 vs 小盘 |
| 非线性市值 | 中盘股效应 |
| 杠杆 (leverage) | 高杠杆 vs 低杠杆 |
| 账面市值比 (book_to_price) | 价值因子 |
| 成长 (growth) | 增长预期差异 |
| 动量 (momentum) | 过去表现延续 |
| 盈利能力 (earnings_yield) | 盈利收益差异 |
| 贝塔 (beta) | 市场敏感度 |
| 残差波动率 (residual_volatility) | 剥离市场风险后的波动 |
| 流动性 (liquidity) | 交易活跃度差异 |

**解读**:
- 暴露差值 = 组合暴露 - 基准暴露（hs300）。差值越接近 0，说明对该因子风险中性，策略收益稳定性越好
- 远离 0 说明该因子上有主动暴露，可参考因子定义调整持仓以趋近基准

## 收益详情

| 项目 | 说明 |
|---|---|
| Backtest | 回测收益 |
| 国家因子 | Barra 模型中横截面线性回归的截距项，因子值恒为 1，代表大盘整体收益 |
| 风格/行业因子收益 | 由对应因子暴露带来的收益（共 10 个风格因子 + 11 个行业因子 jq_l1） |
| 特殊收益 | 剔除已知可解释因子收益后，其他不可解释的收益 |

## 持仓分析

| 分析项 | 说明 |
|---|---|
| 前 10 大持仓 | 最大持仓股票 |
| 持仓收益 | 各股票持仓收益，其中内部收益率为一项投资可望到达的报酬率 |
| 日交易股数 | 每日交易股数的绝对值之和 |

## 常见问题

- 归因分析是回测后自动生成的，不需要在代码中调用
- 基准默认沪深300，与 `set_benchmark` 设置的基准可能不同
- 风险指标只在每天 17:00 更新，日内不更新
- Brinson 归因中基准权重按沪深300行业权重计算
- 所有风险因子暴露的基准都是 hs300 指数，同一时间段下暴露度一致
